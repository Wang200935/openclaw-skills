# Remote AI integration lessons

- Remote AI should enrich diagnosis, not be the only thing keeping the guardian useful.
- Keep fallback local heuristics available.
- Redact secrets before logs, reports, and UI rendering.
- Retry transient upstream failures, but do not hammer non-retryable blocks.
- Verify the remote request shape independently before embedding it into the main loop.
