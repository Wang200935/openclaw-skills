Follow rules in this order:

1. **Current official contest implementation plan / template / submission instructions**
2. **Current official format guide or judging guide from the organizer**
3. **Current school-specific submission rules** if the paper is for an internal selection round
4. **This conservative pack** when no higher-priority rule is available

1. Title
2. Abstract
3. Keywords

5. Literature review
6. Method / research process
7. Analysis / findings
8. Conclusion and suggestions
9. References

## Abstract guidance

Conservative default:

- Write **one short paragraph**.
- If no official word-count rule exists, aim for **about 200-300 Chinese characters**.
- Include four elements in compressed form:
  1. research topic/problem
  2. method or material
  3. main finding
  4. conclusion/value

- citations in the abstract unless the template requires them
- long background history
- empty claims like "?Ęµ??ŹĶ??āĶĆāÕā╣??
- results you never show later

## Keywords guidance

- Use **3-5 keywords**.
- Prefer nouns or noun phrases.
- Put the most central concepts first.
- Do not repeat the entire title word-for-word if better terms exist.

## Body guidance

### Introduction / motivation
Include:

- why the topic matters
- what specific question you want to answer
- scope and object of study
- if useful, a short note on why you chose this case

### Literature review
Include:

- prior studies or traceable sources
- comparison, not just source-by-source summary
- what gap or angle remains for your paper

Minimum safe move: summarize several relevant sources, compare them, then state what your own paper adds.

### Method / research process
State clearly:

- what materials/data you used
- how you collected them
- how you compared, categorized, counted, interpreted, or analyzed them
- obvious limits in the method

### Analysis / findings
- Show evidence before conclusions.
- Separate your observations from borrowed claims.
- Use subheadings if the section gets long.
- If page space is tight, keep this section stronger than literature padding.

### Conclusion and suggestions
- Answer the research question directly.
- Summarize the main finding in plain language.
- State limits honestly.
- Add suggestions only if they come naturally from the evidence.

## References and citation guidance

Use conservative rules drawn from the local research note:

- Every citation in the body should point to a traceable source.
- Only list works in the references that were actually used in the paper.
- Prefer **at least 3 references**.
- Do **not** rely entirely on web-only casual sources.
- Avoid untraceable forum, Q&A, or chat-style sources unless a rule explicitly allows them.
- Do not fabricate page numbers, authors, dates, or publishers.

### Quotation safety rules

- Direct quotations should be clearly marked.
- Paraphrases still require citation.
- If a contest rule mentions a direct-quote length cap, obey it.
- Based on the local research note, a conservative assumption is that **a single directly quoted passage should stay short**, and **50 Chinese characters is a prudent upper bound** unless an official rule says otherwise.

### Citation format uncertainty

If the contest gives a specific citation style, use it exactly.

If no fixed style is given:

- Keep one consistent style throughout.
- At minimum, make author/source name, year/date, title, and retrieval/publication info clear enough to trace.
- Consistency matters more than mimicking a university style badly.

## Page budgeting

When no better rule is known, design for a competition-safe length of **about 4-10 A4 pages**.

Because some sources disagree on whether the cover page counts, use this conservative approach:

- Aim for the **main paper itself** to fit comfortably within the limit.
- Do not pad to the maximum.
- If a cover page is required, keep enough margin so the total still stays safe under the organizer's rule.
- If a submission system or official PDF explicitly says the count includes or excludes cover pages, obey that instead.

### Suggested budget within a 4-10 page paper

- Abstract + keywords: 0.5 page
- Introduction: 0.5-1 page
- Literature review: 1-2 pages
- Method: 1-2 pages
- Analysis / findings: 1-3 pages
- Conclusion: 0.5-1 page
- References: as needed

If you must cut length, cut repetition first, especially in background and literature review, before cutting method or findings.

- English: **Times New Roman 12ń┤?*
- black text
- single line spacing
- margins around **2 cm** on all sides
- page header with title
- centered page number in footer, small size

## Handling font/size uncertainty safely

Use this decision rule:

- If the current official rule gives exact font/size/layout, copy it exactly.
- If the contest is likely the national high-school small-paper contest and no newer rule contradicts it, the above settings are the safest default.
- If the school provides a template file, use the template even if the font differs.
- If nothing is known at all, use a readable 12pt-equivalent Chinese font, clear headings, and consistent spacing, but mark this as a fallback rather than official policy.

## Cover page uncertainty

Local sources may conflict on cover-page handling.

So use this conservative rule:

- **If the official organizer provides a cover format, follow it.**
- **If the official submission rule says no cover or says uploading a cover causes disqualification, do not include one.**
- **If there is no clear instruction, check the current organizer notice before final export.**

Do not assume that old school examples override current contest submission rules.

## Identity and compliance safety

- Remove student names, class numbers, school identifiers, acknowledgements, or file metadata if the contest requires anonymous review.
- Do not include self-identifying content in appendices if anonymity is required.
- If the current rule set restricts AI-generated content, follow that rule strictly.

## Practical conservative checklist

Before submission, verify:

- title is specific and matches the actual study
- abstract is short, factual, and aligned with the paper
- keywords are 3-5 items
- body sections follow the required order
- citations appear wherever ideas or words came from sources
- references are traceable and actually used
- length stays safely within the official page rule
- font, size, spacing, margins, header, and page numbers match the active template
- no cover-page mistake under the current rules
- no identifying information if anonymous judging is required
