# Plugin install order

1. Read README / plugin manifest / config schema.
2. Identify mandatory config keys.
3. Install plugin files.
4. Add plugin config and trust/load settings.
5. Restart gateway.
6. Verify plugin registered.
7. Verify plugin functionality with a real command.

Do not restart first and hope to discover missing config later.
