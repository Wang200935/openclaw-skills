# Format Guide

Use this conservative small-paper format when no official contest template is provided.

## Recommended layout
- Title
- Abstract: 200-300 Chinese characters or one short paragraph
- Keywords: 3-5 terms
- Main body: 5 sections max
- References: enough to support claims, preferably traceable

## Conservative formatting defaults
- Font: common readable Chinese font such as ??? or ???? if required by school; otherwise a standard readable Chinese serif/sans font
- Body size: 12pt equivalent
- Line spacing: 1.5 or platform default if a template exists
- Headings: consistent and hierarchical
- Page count: prefer concise competition-ready length; if no rule is known, design for roughly 4-10 pages of main content

## Hard rules
- Follow the official organizer template over this guide when available
- Do not invent page/word limits as official policy
- If a contest provides a cover, abstract box, or numbering scheme, mirror that exactly
