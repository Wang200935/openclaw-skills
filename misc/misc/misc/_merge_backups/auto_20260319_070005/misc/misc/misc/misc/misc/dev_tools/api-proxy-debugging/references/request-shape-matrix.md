# Request-shape matrix

For each endpoint, record:
- method
- URL
- headers
- body shape
- status code
- success body excerpt
- failure body excerpt

Maintain a known-good example for:
- `/models`
- `/chat/completions`
- `/responses`

This prevents repeating the same blind probing later.
