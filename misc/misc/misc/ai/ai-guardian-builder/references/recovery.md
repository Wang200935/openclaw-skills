# Recovery flow

1. confirm unhealthy state
2. collect snapshot
3. generate diagnosis
4. try soft recovery first
5. verify
6. try stronger recovery
7. verify
8. write incident report

Do not jump directly to the strongest action unless there is a clear reason.
