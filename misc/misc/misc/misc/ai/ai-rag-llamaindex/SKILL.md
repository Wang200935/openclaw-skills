# RAG / LlamaIndex-style Systems

Use this skill for document-centric AI systems.

## Core workflow

1. Load documents deterministically.
2. Normalize and chunk carefully.
3. Build or query an index.
4. Retrieve only relevant context.
5. Synthesize with citations or explicit source traces.

## Rules

- Distinguish ingestion bugs from retrieval bugs.
- Verify documents are actually loaded before blaming the model.
- Prefer small validation corpora first.
- Log chunk counts, source paths, and retrieval hits.
