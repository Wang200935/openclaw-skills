# Streamlit-style AI UI

Use this skill when a quick local control panel is needed.

## Good defaults

- Show current health first.
- Expose recent actions and logs.
- Keep diagnostics readable.
- Add manual action buttons only for safe operations.
- Prefer local dashboards for observability before adding remote delivery.

## Typical panels

- current health
- last diagnosis
- recent events
- latest report/logs
- manual repair actions
