# Citation Hygiene

## Rules
- Only cite sources you can trace.
- Keep author / title / year / link or publication info consistent.
- Do not invent page numbers.
- If you used a webpage, record access date when needed by the target format.

## Safe practice
- Keep a raw source list while writing.
- Match in-text claims to actual source content.
- If a claim came from your own dataset, do not attach a fake citation.
