# Section Templates

## Title
Use a concrete problem + method/object + scenario if possible.

## Abstract template
- Research motivation in 1-2 sentences
- Data/source or object of study in 1 sentence
- Method in 1-2 sentences
- Main finding in 1-2 sentences
- Final implication in 1 sentence

## Introduction / motivation template
1. Why this topic matters now
2. What exact problem you focus on
3. Why existing tools/research are not enough
4. What this paper actually does

## Literature review template
- Group prior work by theme instead of source-by-source summaries
- End with the gap your paper addresses

## Method template
- Research object / dataset
- Collection method
- Cleaning / preprocessing
- Analysis / model / system steps
- Evaluation method

## Findings template
- Show the result first
- Then explain what it means
- Then compare with expectation or prior work

## Conclusion template
- Restate the problem
- State the most defensible findings
- Admit limitations
- Give realistic future directions
